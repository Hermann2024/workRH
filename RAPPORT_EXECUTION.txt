1. frontend/angular-app/src/app/pages/pricing-page.component.ts
   
   Erreur détectée:
   - TS2367: Comparaison incompatible entre HttpErrorResponse et TimeoutError
   - Ligne 136: if (error?.name === 'TimeoutError' || error?.message?.includes...)

   Correction appliquée:
   - Changement du type: HttpErrorResponse → HttpErrorResponse | unknown
   - Ajout d'une vérification instanceof HttpErrorResponse
   - Gestion correcte du cas TimeoutError avec un if/else
   - Résultat: Code type-safe, pas d'avertissements

2. frontend/angular-app/src/app/components/checkout-loading.component.ts

   Erreur détectée:
   - onCancel n'était pas déclaré correctement comme @Output()
   - Utilisait un objet `as any` (mauvaise pratique)
   - Ligne 216-218: onCancel = { emit: () => {} } as any

   Correction appliquée:
   - Ajout des imports: Output, EventEmitter
   - Conversion: @Output() onCancel = new EventEmitter<void>()
   - Résultat: Type-safe, suivit les conventions Angular
